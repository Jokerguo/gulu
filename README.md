## 介绍

学习Vue的过程中做的一个UI框架，希望对你有用

## 开始使用

1. 安装
使用本框架前，css设置border-box
```
*,*::before,*::after{box-sizing: border-box}
```
IE８及以上浏览器都支持此样式


2. 安装 gulu
```
    npm i --save gulu-testing
```

3. 引入 gulu

```
    import {Button} from 'gulu-testing'
    import 'gulu-testing/dist/index.css'

    export default {
      name: 'App',
      components: {
        'g-button' : Button
      }
    }
```
